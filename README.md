# VibeColorizer

Автоматически подстраивает цвет анимации «Моей волны» под обложку текущего трека с помощью фильтров.

## Как работает
Аддон извлекает доминантный цвет из обложки трека, вычисляет разницу с исходным цветом Волны и применяет фильтры `смещения цвета`, `насыщенности` и `яркости` к анимации Волны.

## Совместимость
- Работает только с новой страницей «Моя волна» (эксперимент `WebNextNewWaveTab`).
- Не конфликтует с другими аддонами.
- Имеет смысл использовать только с отключенной настройкой `Использовать альтернативную цветовую схему Волны`
